# pastor_bot
## TL;DR
あなたの悩みについて、聖書に基づいて相談に乗ってくれます。

## 機能について
| スラッシュコマンド | 機能 |
| --- | --- |
| `help` | 本botの利用説明を表示 |
| `christianity` | 聖書に基づいて、あなたのお悩み相談をします[学業、金運、恋愛etc…]。 |
| `islam` | コーランに基づいて、あなたのお悩み相談をします[学業、金運、恋愛etc…]。 |
